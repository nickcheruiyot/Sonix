# Sonix 🎵

Modern Offline Music Player built with Kotlin and Jetpack Compose.

## Features
- Offline music playback
- Albums section
- Artists section
- Song ordering
- Mini player
- Modern Material UI
- Clean MVVM architecture
- Jetpack Compose UI
